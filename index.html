Paideuo Task Manager

Paideuo is the Greek word for maturity through guidance and discipline. Paideuo Task Manager is a task manager which takes on this spirit. It allows you to create categories and tasks. For every task you create you can assign an accountability partner, who receives an email if you fail to complete the task by the set date.
